# テーマの説明

「Kirameki」はシンプルかつ軽量なブログ用WordPressテーマです。さまざまなショートコードやウィジェットを利用することで柔軟なカスタマイズが可能となっています。
レスポンシブウェブデザインに対応しているのでスマートフォンでも快適に閲覧することができます。
コーディング初心者なのでコードがガバガバですが、温かい眼で見てくれるとうれしいです。
さらに詳しい情報は (http://gadgetone.net/) でご確認ください。
